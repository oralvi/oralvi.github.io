玩完orwell有点话想说

发表于 2020-06-02 分类于 个人随笔
本文字数： 942 阅读时长 ≈ 1 分钟

今天晚上绝赞摸鱼并玩通了orwell。

这些文字可能码的质量很差，请大家随意

我发觉了自己身上的那股微妙的三观相当诡异。我在“他们”没有与我谈话时，我就只是一个冷漠的旁观者，或者说是监视者。我看着他们所做的一切，我站在国家这边，为他们的所做所为感到不满，觉得他们就是秩序的破坏者，是罪犯，我甚至在最后可上传数据块有限的情况下推出了哈里森的所在位置，并明白了朱丽叶是这些事件的主谋。。但最后我选择包庇“思想”，举报上司，使得orwell“被销毁”，自己被通缉。我并没有坚定地站在任何一方的位置上，处于所有立场的中间（我是这么认为的），我到最后举报上司前我还觉得朱丽叶为什么不该为这些问题戴上镣铐，而是哈里森在我的情报下成了替罪羔羊。。但我也明白我一直是如此。就像我们现在“网络并不是法外之地”一样，奥威尔的假想或许越来越接近了。诚然这种监视我是认为它是有好处的，毕竟用这种工具“让群众们得以安眠”也是不错的，但？所谓的隐私就消失了，你活在眼睛的注视下，想想就毛骨悚然。我大概是有这种感觉了。我知道这些感觉不是凭空得来的，而是我感受到的，大手是在收紧着的，咱们的嘴巴也最好得收紧点。所以我每次在墙内看到有些人的“失智”言论是既表示理解又表示痛恨。相当矛盾的感觉，对，没错，大家都应该有言论自由，但是，在现有国情的情况下去说一些不符合的话语，我看这不是-1s行为吗。但是我又认为，适当的言论管控是必要的，机器能够正常运转也是需要合适的零件装配一起才行的啊。。所谓的自由又是什么呢？我现在还是不明白。

高中看了不少书，《1984》《动物庄园》，以及一些国内作家写的对自由探讨的文字也是看了不少，或许当初所积攒的知识可能比我现在积累的要多得多（现在脑子里就装着一些没用的东西，没好好学习），但当初的我的思想一定是更加开放而又激进的，不像现在，觉得活着就像在啃鸡肋一样，没有所谓的动力去学习新的知识，就这样一天天的沉沦。。我并没法像那些魏晋时代的文人一样看透人生，而是觉得看了没用，得过且过。

但如果真的想要追求自由的话。。其实我一直在追求着自由。我之前就一直希望我能够离开这里，到那里去，包括我复写多次，到现在仍未成形的小说一样，也在反复强调着这些话语。但我已经麻木了？可能还没，我这该死的/

他们的青春恋爱物语，与我

发表于 2020-02-29 更新于 2020-03-01 分类于 个人随笔
本文字数： 1.5k 阅读时长 ≈ 1 分钟

这是我第四次（大概）完整地（也不算是很完整地）看完《我的青春恋爱物语果然有问题》（后面简称春物，不然名字实在是太长了w）。能够看到他们的圆满结局（至少对于我来说已经足够圆满），我很满足了。

其实之前也有写过类似的文字，但，似乎和之前的感觉不太一样了。最开始看是初三的时候，将要升上高中，也有着和大老师类似的黑历史，不禁有了“大老师是我学习的榜样”的感觉。现在回想起来我可是真的自负。怎么说呢，明明不可能完全一样的我，缺乏足够的洞察能力的我，可能和大老师唯一类似的地方就仅仅是对某种感情迟钝罢了（但现在看来也并不迟钝，八幡只是看破而不愿说破而已）。反正我觉得我度过了相当荒唐的一段高中时光。相比大老师的所谓过剩的自我意识，我是更加过分地，自己做过的事情只是为了欺骗自己，让自己觉得“啊啊，这样也不错”的错觉。高中的时候也有断断续续的回看，但感觉仍然没有太大的变化。

大概是第三次吧，我是在春物14卷发售后，突然想回忆一下这个故事所以就再次拾起，并买回简中的书回来收藏。当然我的想法仍然幼稚，并开始擅自期待着能出现奇迹，这些已经超出范围的想法到现在仍然困扰着我。

希望能够得到真物，但这只能是一种奢望。

「换个问法。……你找到你的真物了么？」

这次得出结论并没有花费太长时间。因为老师曾经教导过我。

去思考、去挣扎、在烦恼中喘不过气……我的回答早已决定好。所以，我把我的头又撞了回去，扬起嘴角微微一笑。

「谁知道呢。这可不是那么容易就能找到的东西呢。」

「要是听到了可是会被发火的哦。又或者会躲在看不见的角落里悄悄抹眼泪也说不定呢」

「真麻烦……请别说那么现实的话……话说回来，这是在说谁啊，才不是那样呢」

「是嘛，或许确实不是那样吧」

平塚老师一边晃动肩膀大笑着，一边使劲挪了挪腰，坐到了我的旁边。

「若是对一个女生抱有共鸣、亲昵、好奇、感同身受、尊敬、嫉妒，以及在此之上的感情的话，那么单用『喜欢』来描述肯定是不够的」

平塚老师盘坐着用手托着下巴，一边用手指一根一根地数着刚才列举的感情，一边目不转睛地盯着我。

「所以，才会无法分别，无法离散，就算产生了距离，就算时光流逝，仍会互相吸引……这或许称得上是真物」

「是这样吗？我不太清楚啊」

我耸了耸肩，露出了讽刺的笑容。

肯定，永远，都不会知道我们的选择是否正确。或许现在依然在犯错也说不定。

然而，就算旁人突然塞给我唯一的正确答案，我大概也不会承认吧。

「所以，我会一直怀疑下去。大概，无论是我还是那家伙都不会如此轻易地相信」

「虽然离正确答案还很遥远，但你的答案能得100分满分。你真是一点也不可爱啊……正因如此，你才是我最棒的学生」

这直到我第四次看，才慢慢察觉到。其实我心底里也很清楚，这种宝贵的感情，不是轻易就能够获得的，它需要时间的沉淀和磨合。虽然觉得贬低自己并不是什么好行为，但这就是事实。不擅长读空气的，莽撞的，自傲的，弱小的我，在期待着很难的东西。

近一段时间情绪挺低沉的，什么事也不感兴趣，什么事也不想做。但，我是否曾经期待过，这种事情，有没有那个必要，我也不太清楚。要是有谁，能够帮我一把，就好了。虽然我觉得不可能。心伤都是自己治疗的。可以说，我并没有那么扭曲，也没有承担一切的自信，现在看来，我就是一个逃兵吧。我这几年看过的故事，更多更多的，是各种巧合。毕竟一介凡人，怎么会遇到奇迹。

我也不知道，我什么时候才能再次站立。

似乎越写越偏移本来想讲的了。但我每次都是这样，因为摇摆不定的情绪，总在情绪低谷时码字，而感觉想要表达出来时又往往没得记录下来。而我又对这类文字没有记忆感，写着写着就变成发牢骚一样的东西了。。

可能是我自己的痴心妄想吧。

现在我，我也想和八幡一样。能够找到真物。

但估计短时间是不可能啦。

那段逐渐复苏的记忆，又再次散去

发表于 2019-11-18 更新于 2020-03-01 分类于 个人随笔
本文字数： 941 阅读时长 ≈ 1 分钟

之前写的春物感想，想想还是搬一下。。。

一直很想写点和春物有关的感想，之前说说里也是再看完动画后有发表一些现在看来十分幼稚的说说。到现在已经找不到了。因为，我和我自己，已经将说说删除了，连带着那段高中时的各式各样的，欢笑着的，悲伤着的记忆。
我也是今天得知春物的结局出来了。是雪乃，和大老师一起了。

我从一开始就是雪乃党，所以雪乃的胜利，我很开心，而且我一直感觉，雪乃最后一定能够和大老师在一起。

耳机里放着”エブリデイワールド”，我也开始回想起，当初的情景。春物是我的入坑番，准确来说，是我的对动画，对二次元的入坑作。它曾躺在我的收藏夹里两个多月之久，在当初上破站只是为了看猫和老鼠鬼畜视频，和为了了解当初喜欢的人的喜好的我看来，春物并不好看，只看了短短的开头，就觉得这种压抑的青春不适合我。但没想到的是，接下来的我，活成了半个大老师的样子。

我是在初三升高中的暑假闲来无事便看起春物来的。在那个熬夜并不习惯的时候，我冒着被老爸收走手机的风险，经过努力终于在某个半夜将它看完了。记得当初凌晨四点发了一条说说，具体是说什么我也记不太清了。反正与春物有关就是了。

当时的我啊，很羡慕大老师，也很憧憬大老师，被女孩子喜欢是哪个青春期男生不乐意的呢？更何况是对我这种妄想美好恋情的我来说，是很开心的。说实话上了高一，也补了不少后宫题材的番剧，但印象最深的，最受我所认可的，还是春物。我曾有一段时间，差点将这种想法套到现实中，那时或许也是我最开心的一段时光吧。有可靠的异性聊天对象，有着快乐的生活。现在看来，估计我是只记得好的，而忽略掉那些可能发生过的杂音了吧。说我活成半个大老师是一种自嘲。因为我学会了带上灰色眼镜看着这个世界，对未来抱着迷茫与悲观，而没有学会最重要的，温柔，和沟通。（可能还有很多很多没有学会的对我有益的东西）

正如我现在，我对自己不满，正是因为，我只对他人的高要求而对自己放松的双标以及对自身的不够喜爱。连自己都不能够喜爱的人，怎么会对他人喜爱呢？

春物中最重要的，是羁绊，以及三人之间的关系。这种东西我仍然处理不好。

或许我刚刚还有许多话想说下去…

但是，我还是没有勇气直接盖上这段记忆…

我难以面对放下困境，我也想要有一种连接。

但，我也许只是在一个人，说着随意的话，随意的活着吧。
