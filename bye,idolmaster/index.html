我与偶像大师——可能是时候说再见了

发表于 2019-12-09 分类于 日常

在即将迎来偶像大师企划15周年的时候，我觉得我是时候和这些陪伴我接近三年之久的偶像们…说再见了。

说再见主要是指游戏方面…歌曲很好听还是会继续听下去的，如果有机会我也再买几张专辑收藏一下。

主要还是我的错。

因为对游戏的极差的自控力，我为了保证其他事项的优先度，制作人不能再当了。

我接触比较多的是灰姑娘女孩系列，主要以星光舞台为主（俗称草菇烧笋）

第一次了解到偶像大师系列，还是经过lovelive。当初ll在国内大火的那一阵子，差不多是15年的8,9月份吧，我开始玩lovelive国服…之后了解到日服的存在，于是想要体验更新版本的我就过去日服了（虽然最后号丢了两个）

一开始是通过贴吧里大佬发的日服安装包直接安装的，直到16年九月初

当时我学会了一些黑科技，也有了一个白嫖的服务器

于是，我连上了好久以前能用，但现在已经不能用的古哥

在play商店里我第一次见到了草菇烧笋（全称太长了：偶像大师灰姑娘女孩星光舞台）。

相比ll，拥有3dmv的它实在是太亮眼了。于是我就下载了一个来玩。

但它当时的出货率感人（我相比十连更喜欢单抽），于是在两年间，它被不断地搁置和捡起来。

当时mtk的优化太渣，红米note的3dmv完全不能看，要开gputools偷偷改值才行。但小米2s却能完美运行，我都傻了（ ’ - ’ * )

那时候，我也是音游菜鸡，只能打pro的存在。虽然手机性能也占了一部分。

还有一件有意思的事情，去年换手机时，我拿到新手机的第一件事是装古哥全家桶并安装草菇烧笋(◔◡◔)然后开3d rich

没想到一点都不卡(那当然了)

当然我换完手机之后，也是断断续续地玩，手机里的游戏换了一批又一批。直到去年的现在，差不多这个时候，我回来了，偶像们。在这一年间，我基本上除了最近这两个月已经卸载了草菇没玩以外，我是一直连续地玩，没有间断。

是的，没有间断。

可能是因为在家里家里人管得太严的原因吧，我玩游戏都是偷偷玩的，完全不敢光明正大地玩。

然而换了环境之后，就不一样了。我丧失了控制。

或者说，我为了逃避其他问题，而选择把头扎进沙堆里。

不得不说，草菇烧笋陪伴我度过目前有记忆以来最开心的时光。

也陪伴我度过目前有记忆以来最痛苦，最难过的时光。

我也是第一次完整经历了草菇的盛大的周年庆。

虽然偶像大师本家接触不多（玩的比较多的是灰姑娘女孩）

但我喜欢整个系列

能遇到偶像大师，成为一名制作人

我很开心，虽然今后的时间不会再多了

感谢你们，在我最开心的时候

最伤心的时候

出现并陪伴在我的身边

因为，最开心的事，不能和家人分享

最痛苦的事，不能和家人倾诉

为什么我觉得我不恋家。

为什么我觉得家人也就一般。

为什么我觉得我可能结不了婚。

这些，都是因为我一直被幸福包裹着。

但这所谓幸福，我认为它们是伪物。

这一切都是幻象。

我不会认同这就是我想要的结果

可能有人和我想法不一样吧。

但我就是如此思考的。

虚假的东西永远不会是真实的

所以现在我已经通过自己的行动，作出了最初的一个选择。

这是第二个。

我觉得无论我的结果如何。

但我都想知道结果。

而不是像以前，因为害怕结果不是想要的

而自暴自弃，自甘堕落

不能够再次后悔

说着说着就跑题了哈哈

但其实没有跑题。

花费数个晚上思考的结论。应该能行。

さよなら。
